/ 02 · ABOUT

운영되는 시스템을 만드는 엔지니어.

서울에서 활동하는 AI 엔지니어입니다. 프로토타입에서 멈추지 않고, 실제 업무 흐름에 들어가 매일 돌아가는 시스템을 만드는 데 집중합니다. LLM Agent 설계부터 데이터 파이프라인, 클라우드 배포와 운영까지 한 번에 책임지고 만듭니다.

제가 만드는 시스템은 관찰 가능성(observability)을 기본으로 합니다. 장애가 났을 때 어디에서 무엇이 멈췄는지 추적할 수 있어야 하고, 중요한 의사결정에는 사람이 개입할 수 있어야 한다고 믿습니다(Human-in-the-Loop). 데모에서는 화려해 보이지만 현장에서 깨지는 시스템을 많이 봐 왔기 때문입니다.

API 비용과 운영 비용을 설계 단계에서 먼저 따지고, 실패했을 때의 비용까지 계산에 넣습니다. 그래야 오래 돌아갑니다.
