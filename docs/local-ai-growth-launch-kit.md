# Local AI Growth Launch Kit

소상공인 대상 AI 운영 자동화 대행을 빠르게 팔기 위한 실행 문서입니다.

## 1. 한 문장 제안

네이버 플레이스, 리뷰 답변, 블로그 초안, 이벤트 문구, 고객문의 정리를 매주 대신 처리해드립니다.

## 2. 가장 먼저 노릴 업종

1. 미용실, 네일샵, 피부관리실
   - 리뷰와 재방문이 중요하고, 사장님이 콘텐츠를 꾸준히 만들기 어렵습니다.
2. 카페, 디저트샵, 식당
   - 메뉴, 시즌 이벤트, 방문 리뷰를 계속 콘텐츠로 바꿀 수 있습니다.
3. 필라테스, PT샵, 학원
   - 상담 문의와 후기 관리가 매출로 바로 이어집니다.
4. 병원, 치과, 한의원은 나중에 접근
   - 단가가 높지만 광고 규정 검토가 필요해 초반 실험 대상으로는 느립니다.

## 3. 판매 패키지

### Starter: 월 30만원

- 리뷰 답변 초안 주 2회
- 네이버 플레이스 개선 체크리스트 월 1회
- 블로그 초안 월 4개
- 월간 실행 리포트 1개

목표: 첫 고객을 쉽게 설득하기 위한 진입 상품.

### Growth: 월 70만원

- 블로그/인스타 초안 월 12개
- 리뷰 불만/칭찬 요약
- 시즌 이벤트 문구 4개
- 경쟁 매장 3곳 비교 리포트
- 주간 실행 체크리스트

목표: 실제 월 운영 매출을 만드는 주력 상품.

### Automation: 별도 견적

- 카카오/이메일 반복 문의 답변 봇
- Google Sheets 매출/문의/콘텐츠 대시보드
- 주간 자동 리포트
- 업종별 FAQ 지식베이스

목표: 월 구독 고객에게 추가 매출을 붙이는 확장 상품.

## 4. 무료 진단 리포트 구성

무료 진단은 1페이지로 짧게 보냅니다.

1. 현재 플레이스 첫인상
2. 리뷰에서 반복되는 칭찬 3개
3. 리뷰에서 반복되는 불만 3개
4. 블로그/인스타 콘텐츠로 바꿀 소재 5개
5. 이번 주 바로 고칠 액션 5개
6. 2주 파일럿 제안

## 5. DM 문구

### 인스타/카카오 첫 메시지

안녕하세요. 성남에서 AI 자동화 작업을 하는 Adam입니다.

네이버 플레이스와 리뷰를 보다가, 사장님 매장은 콘텐츠로 더 살릴 포인트가 많아 보여서 연락드렸습니다.

제가 무료로 1페이지 진단 리포트를 만들어드릴 수 있는데요. 리뷰에서 반복되는 장점/불만, 블로그 소재, 바로 고칠 플레이스 항목을 정리해서 보내드립니다. 받아보시고 필요 없으면 편하게 넘기셔도 됩니다.

괜찮으시면 매장명 기준으로 진단해서 보내드릴까요?

### 무료 진단 발송 후

사장님, 말씀드린 무료 진단 리포트 정리해서 보내드립니다.

핵심은 세 가지입니다.

1. 리뷰에서 반복되는 강점은 콘텐츠로 아직 충분히 쓰이지 않았습니다.
2. 플레이스 설명/사진/소식란에서 바로 고칠 항목이 있습니다.
3. 블로그 소재는 최소 10개 이상 바로 만들 수 있습니다.

원하시면 2주 파일럿으로 리뷰 답변, 블로그 초안, 이벤트 문구까지 제가 직접 운영해보겠습니다. 파일럿 비용은 15만원으로 잡고, 결과 보고 계속할지 결정하시면 됩니다.

## 6. 전화/미팅 질문

- 매장 신규 고객은 주로 어디서 들어오나요?
- 네이버 플레이스 관리는 누가 하나요?
- 리뷰 답변은 얼마나 자주 하시나요?
- 블로그나 인스타를 꾸준히 못 하는 이유가 시간 때문인가요, 소재 때문인가요?
- 가장 팔고 싶은 메뉴/시술/수업은 무엇인가요?
- 이번 달에 밀고 싶은 이벤트가 있나요?

## 7. 7일 실행 계획

### Day 1

- 타깃 업종 하나 선택: 네일샵 또는 미용실 추천
- 주변 30개 매장 리스트업
- 매장별 플레이스 링크, 인스타, 연락처 정리

### Day 2

- 5개 매장에 무료 진단 샘플 작성
- 공통 템플릿 만들기

### Day 3

- 20개 매장에 DM 발송
- 답장 온 곳은 24시간 안에 진단 리포트 제공

### Day 4

- 추가 20개 매장 발송
- 첫 미팅/통화 제안

### Day 5

- 2주 파일럿 제안: 15만원
- 파일럿 범위는 리뷰 답변 10개, 블로그 초안 2개, 이벤트 문구 2개

### Day 6

- 첫 파일럿 납품물 제작
- 납품 전후 비교 이미지를 남겨 포트폴리오화

### Day 7

- 결과 리포트 발송
- 월 30만원 Starter 또는 월 70만원 Growth로 전환 제안

## 8. 운영 도구

- 고객별 폴더: 매장명, 링크, 리뷰 캡처, 리포트, 콘텐츠 초안
- Google Sheets: 고객 목록, 연락일, 상태, 다음 액션
- AI 프롬프트: 리뷰 요약, 답변 초안, 블로그 초안, 이벤트 문구
- 납품 방식: Google Docs 또는 PDF 1페이지

## 9. 첫 달 목표

- 무료 진단 20개 발송
- 통화 5건
- 2주 파일럿 2건
- 월 구독 전환 1건

첫 고객 목표는 큰돈이 아니라 실사용 사례입니다. 한 곳이라도 월 운영을 만들면, 그 결과물을 다음 영업 자료로 씁니다.
