Save Backup and Recovery

Export the save on the old device, then enter the entire generated text into Import Save on the new device.

Use Restore Pre-Import Backup to return to the state immediately before an import. Exporting before deleting the app is recommended.

Game Center/iCloud Sync Issues

Enable automatic sync in Settings, then confirm that Game Center is signed in and iCloud Drive is enabled. If sync stalls, check the network and press Sync Now again.

If two save candidates appear, compare their scene, funds, retirements, and save time before choosing one. The pre-conflict backup can restore the state from before the choice.

Reporting a Problem

Use Copy Diagnostics in Settings to include recent errors and progression-stall information. Diagnostics do not leave the device until you share them.
