개인정보 처리방침

시행일: 2026년 7월 17일

개요

나라먹기는 별도 개발자 계정을 만들지 않습니다. 기본 게임은 로그인과 광고 시청 없이 오프라인으로 플레이할 수 있습니다.

공작 패널에는 사용자가 직접 선택하는 선택형 보상 광고만 있으며 배너, 자동 전면 광고와 앱 실행 광고는 없습니다.

선택형 보상 광고

선택 광고는 로컬 날짜 기준 하루 최대 5회입니다. 보상은 민심 보석 5개, 민심 보석 5개, 자동 재산 15분, 무료 성장 1회, 자동 재산 30분 순서입니다. 광고를 보지 않아도 모든 핵심 진행, 캐릭터, 도구와 엔딩을 획득할 수 있습니다.

Google Mobile Ads SDK와 User Messaging Platform(UMP)은 광고 제공, 측정, 동의 관리와 부정 사용 방지를 위해 IP 주소, 대략적 위치, 기기 식별 정보, 광고 데이터, 광고·영상 상호작용, 크래시 로그와 성능·진단 정보를 처리할 수 있습니다.

광고 요청은 비맞춤형으로 전송됩니다. 앱은 App Tracking Transparency(ATT) 권한을 요청하거나 추적 허용을 보상 조건으로 사용하지 않습니다. UMP가 요구하는 지역에서는 설정의 광고 개인정보 선택에서 선택을 변경할 수 있습니다.

개발자는 광고 이벤트를 별도 개발자 서버로 직접 수집하거나 개인정보를 판매하지 않습니다. Google의 데이터 처리는 Google의 개인정보 보호정책과 사용자의 동의 선택이 적용됩니다.

광고 제거 인앱 구매

광고 제거는 Apple App Store의 비소모성 인앱 구매입니다. StoreKit이 구매 권리를 확인하며 앱과 개발자는 결제 카드 정보를 받지 않습니다.

구매자는 접속한 날 한 번 같은 5회 보상 묶음을 자동으로 받고, 미접속 날짜의 보상은 소급되지 않습니다. 공작 패널에서 구매를 복원할 수 있습니다.

기기에 저장되는 정보

게임 진행, 수집 기록, 음향·햅틱·언어·동작 설정과 진단 기록은 현재 기기에 저장됩니다. 진단 기록은 사용자가 직접 복사하거나 공유하기 전까지 개발자에게 전송되지 않습니다.

선택형 Game Center/iCloud 동기화

사용자가 자동 동기화를 켜고 Game Center 및 iCloud를 사용할 수 있는 경우, 게임 진행 데이터가 플레이어의 Apple 게임 계정에 연결된 Gameplay Content로 Apple의 Game Center/iCloud 저장 공간에 보관될 수 있습니다.

동기화는 앱 기능 제공만을 위해 사용됩니다. 개발자가 운영하는 별도 서버는 없으며 개발자는 동기화된 저장을 수신하거나 보관하지 않습니다. Apple의 데이터 처리는 Apple의 정책과 계정 설정이 적용됩니다.

보관과 삭제

로컬 데이터는 사용자가 설정에서 초기화하거나 앱을 삭제할 때까지 남습니다. 자동 동기화는 언제든 끌 수 있으며, iCloud 저장은 설정의 이중 확인 절차를 거쳐 삭제할 수 있습니다.

앱 삭제만으로 클라우드 저장까지 삭제되지 않을 수 있습니다. 저장 내보내기, 가져오기, 충돌 후보 선택과 가져오기 전 백업 복원 기능도 설정에서 제공합니다.
